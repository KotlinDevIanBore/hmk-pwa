# PostgreSQL Installation Checklist

## Step-by-Step Instructions

### 1. Download PostgreSQL
- [ ] Go to: https://www.enterprisedb.com/downloads/postgres-postgresql-downloads
- [ ] Click "Download" for **Windows x86-64** version (latest version 16.x)
- [ ] Wait for download to complete (~250 MB)
- [ ] File will be in your Downloads folder: `postgresql-16.x-windows-x64.exe`

### 2. Run the Installer
- [ ] Double-click the downloaded file
- [ ] Click "Yes" if Windows asks for permission
- [ ] Click "Next" on welcome screen

### 3. Installation Directory
- [ ] Keep default: `C:\Program Files\PostgreSQL\16`
- [ ] Click "Next"

### 4. Select Components
Make sure ALL are checked:
- [ ] ✅ PostgreSQL Server
- [ ] ✅ pgAdmin 4
- [ ] ✅ Stack Builder
- [ ] ✅ Command Line Tools
- [ ] Click "Next"

### 5. Data Directory
- [ ] Keep default: `C:\Program Files\PostgreSQL\16\data`
- [ ] Click "Next"

### 6. Password (IMPORTANT!)
- [ ] Enter password: **Write it down!**
- [ ] Confirm password
- [ ] Remember: You told me it's "admin" - use this
- [ ] Click "Next"

### 7. Port
- [ ] Keep default: `5432`
- [ ] Click "Next"

### 8. Locale
- [ ] Keep default
- [ ] Click "Next"

### 9. Ready to Install
- [ ] Review settings
- [ ] Click "Next" to begin installation
- [ ] Wait 5-10 minutes for installation

### 10. Completing Installation
- [ ] Uncheck "Stack Builder" (not needed now)
- [ ] Click "Finish"

### 11. Verify Installation
Come back here and tell me: "Installation done"

## Time Required
- Download: 2-5 minutes
- Installation: 5-10 minutes
- **Total: 10-15 minutes**
