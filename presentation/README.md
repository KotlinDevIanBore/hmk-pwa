# Hope Mobility Kenya PWA - Client Presentation

## Overview

This is a comprehensive presentation mockup of the Hope Mobility Kenya Progressive Web Application, designed for client demonstration purposes.

## Features

- 16 professional slides covering all aspects of the PWA
- Interactive navigation (keyboard arrows, buttons)
- Brand-compliant design (HMK colors and styling)
- Visual mockups of all major interfaces
- Workflow diagrams and system architecture
- Exportable to PDF for offline viewing

## How to Use

### Viewing the Presentation

1. Open `index.html` in any modern web browser
2. Use the navigation controls at the bottom or arrow keys to navigate
3. Press spacebar to advance to the next slide
4. Press left arrow to go back

### Exporting to PDF

1. Click the "ðŸ“„ Export to PDF" button, or
2. Use your browser's Print function (Ctrl/Cmd + P)
3. Select "Save as PDF" as the destination
4. Adjust settings:
   - Layout: Portrait
   - Margins: None or Minimal
   - Background graphics: On
5. Save the file

### Converting to PowerPoint

To convert to PowerPoint:

1. First export to PDF (see above)
2. Upload the PDF to an online converter like:
   - Adobe Acrobat online tools
   - Smallpdf.com
   - ILovePDF.com
3. Download the resulting PPTX file

Alternatively, you can manually recreate slides in PowerPoint using this HTML version as a reference.

## Customization

### Updating Content

Edit `index.html` to modify slide content. Each slide is contained within a `<section class="slide">` element.

### Styling

All styles are in `styles.css`. Brand colors are defined as CSS variables at the top:

```css
:root {
    --primary-blue: #0056A6;
    --warm-yellow: #F5E6B3;
    --action-red: #DC143C;
    --dark-gray: #3D3D3D;
    --white: #FFFFFF;
}
```

### Navigation

JavaScript for navigation and keyboard controls is in `script.js`.
